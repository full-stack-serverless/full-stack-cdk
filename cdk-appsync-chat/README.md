## React CDK AppSync Chat

To deploy this app.

1. Clone the repo

2. Change into the `cdk-appsync-chat` directory

3. Install dependencies in main folder:

```sh
npm install

# or

yarn
```

4. Deploy the back end:

```sh
cdk deploy
```

5. Change into the client directory and install dependencies:

```sh
cd chat-client

npm install

# or

yarn
```

6. Run the app

```sh
npm start
```
